# Chat Mirror

## Getting started

Requirements:
- Node.js 20+
- npm 10+

Install and run locally:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the build:

```sh
npm run preview
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
